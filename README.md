# Install atom-config

```
$ git clone https://github.com/johnrlive/atom-config/
$ cd atom-config
$ bash install_packages.sh
```

# Key Mappings:

### Terminal Plus
```
terminal-plus:new =  (cmd + shift + t)
terminal-plus:toggle =  (ctrl + `)
```

### Imdone Atom
```
Open Tasks Board =  (Alt + T)
Todays Journal =  (Alt + J)
```
